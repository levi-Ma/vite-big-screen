# 简易数据大屏

使用 Vue 3+Vite+TypeScript+G2Plot(open-data-plan/g2plot-vue)+Mock(lavyun/better-mock)+axios+naive-ui 搭建的简易数据大屏

## 资源

- [Vue 3](https://github.com/vuejs/docs-next-zh-cn/)
- [Vite](https://github.com/vitejs/vite)
- [TypeScript](https://github.com/microsoft/TypeScript)
- [G2Plot](https://github.com/antvis/g2plot)
- [G2Plot-Vue](https://github.com/open-data-plan/g2plot-vue)
- [Mock](https://github.com/nuysoft/Mock)
- [Better-Mock](http://github.com/lavyun/better-mock/)
- [Axios](https://github.com/axios/axios)
- [Naive UI](https://github.com/TuSimple/naive-ui/)
- [nplayer](https://github.com/woopen/nplayer)
